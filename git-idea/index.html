IDEA使用Git

发表于 2020-07-27 • 字数统计 254

IDEA配置

指定git的安装位置
菜单->settings->Version Control->Git->Path to Git executable: 设置为安装git中所安装的git.exe


指定好后可以点击Test测试

创建并上传项目

搭建好项目后，菜单->VCS->import into Version Control->Create Git Repository->选择项目目录点击OK
这时文件都会变红，右击项目把文件都添加到缓存区

之后红色的文件会变成绿色
上传项目

单击文件可以在下面看版本差异，以及输入提交信息

选择commit&push

单击，url里输入项目地址

点击push

下载项目

菜单->New->Project from Version Control
输入所下载的项目地址

分支

创建分支

菜单->VCS->Git->Branches

根据自己的情况切换分支或者创建分支，我这里创建一个idea分支然后push
刷新页面，分支已经更新上去了

切换分支

idea右下角也可以打开界面，选择Remote Branches下面的分支，然后Checkout
